Consumer Recycling Instructions (On-Pack)

The following recycling instructions will be printed on each SKU's packaging, in a dedicated "How to Recycle" panel (minimum 30 × 40mm):

Glass Jar (100g / 10ml Drops)

1. Rinse jar with water
2. Remove lid/cap
3. Place glass in GLASS recycling
4. Place metal lid in METAL recycling
5. Remove rubber dropper (10ml only) — dispose in general waste

HDPE Jar (250g / 450g)

1. Rinse jar, remove label if possible
2. Remove PP cap
3. Place jar (HDPE #2) in PLASTIC recycling
4. Place cap (PP #5) in PLASTIC recycling
5. Or reuse jar for storage!

Sachet Box / Sachets

1. Flatten outer carton — place in PAPER recycling
2. Individual sachets: currently not recyclable in standard streams
3. Collect empty sachets and return via [Brand Name] take-back program (coming soon)
4. We are working on compostable sachets — launching 2027
