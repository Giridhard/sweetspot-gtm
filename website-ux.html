First Input Delay

Minimize main-thread JavaScript. Defer non-critical scripts. Lazy-load third-party widgets. Use requestIdleCallback for analytics. Target: interactive within 100ms of first input.

▲
CLS < 0.1

Set explicit dimensions on all images and embeds. Reserve space for dynamically loaded content. Font-display: swap with size-adjust fallback. No layout shifts from ads or popups.
